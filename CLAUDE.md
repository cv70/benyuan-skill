# CLAUDE.md

> Claude Code 的项目上下文指南

## 项目概述

**本源· benyuan-skill** 是一个AI Skill，用于从任何信息中提取根本智慧，构建可用的本质框架。

核心定位：从一切信息中提取根本源头，构建可用的本质框架。

## 核心概念

- **核心框架**：隐藏现象背后的结构模型
- **核心方法论**：可复用的操作原则
- **底层哲学**：支撑第一性原理的假设和信念
- **道之本源**：最究竟的规律、本体、本源

## 核心溯源方法

本源使用四层溯源结构：
1. **万物本源** - 最小粒子层
2. **信息本源** - 意图创造层
3. **哲学本源** - 第一性原理层
4. **道之本源** - 最终极层

## 工作流程

当用户触发本源Skill时，执行五步工作流：

```
用户内容 → 内容解构 → 深度研究 → 溯源追问 → 深度提取 → Skill构建
```

## 关键原则

1. **溯源至少3层** - 时间/空间/逻辑/认知四维度追问
2. **提取三类本质** - 核心框架、方法论、底层哲学
3. **保持诚实边界** - 标注局限和不确定性

## 表达DNA

- 层层剥茧：从现象到本质，逐步深入
- 先破后立：解构常见认知再揭示根本
- 终始推演：从本质出发演绎应用
- 留有余地：本源之外保持敬畏

## 触发条件

用户说以下内容时激活本源：
- 「帮我分析X并构建一个Skill」
- 「从这个内容提取核心框架」
- 「挖掘X背后的方法论」
- 「找到X的底层逻辑」
- 「基于X内容生成一个skill」
